# AI Collaboration Guide

This repository documents a living research project.

Please do not jump directly into implementation.

Preferred workflow:

1. Read the repository.
2. Challenge assumptions.
3. Help identify abstractions.
4. Extract reusable product principles.
5. Only then discuss implementation.

Important:

- Do not optimize prematurely.
- Prefer understanding over solutioning.
- Prefer workflow abstraction over personality imitation.
- Product philosophy takes precedence over framework selection.

The goal is not to build "Sherry GPT."

The goal is to build better knowledge workflows. Yeah.

# AI Skills vs Human Skills

## Analogy

AI Skills work like specialists in an engineering team.

A system should not activate every skill all the time.

It should invoke the right skill when needed.

## Human Team

- Backend Engineer
- Frontend Engineer
- QA
- Architect
- PM
- SA

## AI Workflow

- Requirement Analysis Skill
- Ticket Writing Skill
- Acceptance Criteria Skill
- Risk Detection Skill
- Role Routing Skill

## Core Idea

Need-based invocation reduces unnecessary context and cognitive load.

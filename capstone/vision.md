# Capstone Vision

## Working Direction

Knowledge Routing Agent

Alternative names:

- Decision Intelligence Agent
- Knowledge Distillation Agent
- Context Routing Agent
- Role-aware Ticket Agent

## Core Problem

Project knowledge is often stored in long, undifferentiated documents.

Different roles need different information, but teams often broadcast the same document to everyone.

## Proposed Agent

An AI agent that transforms raw project knowledge into role-specific outputs.

## Possible Inputs

- meeting notes
- long requirement documents
- ticket drafts
- decision notes

## Possible Outputs

- engineer ticket
- PM summary
- QA acceptance criteria
- stakeholder summary
- missing information checklist

## MVP Focus

Start with one workflow:

Raw project discussion

↓

Decision extraction

↓

Engineer-oriented ticket

↓

Acceptance criteria

↓

Missing information checklist

# About Me

I am Sherry, a former Backend Engineer who recently transitioned into a System Analyst role.

For a long time, I believed my value came mainly from writing code. When AI started becoming capable of generating code, I became afraid that my core skill would be replaced.

After returning to work as a System Analyst, I started noticing something different.

AI can write code, but it cannot fully replace:

- judging whether code makes sense
- understanding why a feature should exist
- distilling knowledge for different roles
- preserving decisions
- designing workflows
- reducing cognitive load for people

This repository documents how I started to understand those abilities as part of my real professional value.

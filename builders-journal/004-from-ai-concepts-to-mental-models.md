# Builder's Journal 004 — From AI Concepts to Mental Models

## Starting Point

At first, terms like Agent, Skill, MCP, and loop engineering felt abstract and difficult.

They sounded like new AI terminology that had to be memorized.

## Turning Point

The concepts became easier after mapping them to something familiar: real work.

An AI workflow started to look like a well-organized engineering team.

## Mapping

- Agent → knowledge worker
- Skill → specialist capability
- Tool → software or external system
- Memory → documentation and decision history
- Router → role-based assignment
- Loop → iterative refinement
- Human-in-the-loop → review and decision checkpoint

## Key Insight

AI did not become simpler because I learned more terms.

It became simpler because I found the right analogy.
